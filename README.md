# Dog-Ceo
Aplicación Android que permite listar razas-subrazas de perros y mostrar imágenes asociadas.

API utilizada: [Dog API](https://dog.ceo/dog-api/).

Nivel de API Min: 21 - Android 5.0 (LOLLIPOP)

CompileSdk: 32
